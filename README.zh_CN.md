# 月落星河的国际化个人网站

月落星河的全新国际化个人网站采用 Vue 3 开发，并尝试了许多我以前从未尝试过的新功能，例如多语言支持、自定义Dialog、页面过渡动画、监听鼠标滚动与屏幕滑动事件切换路由、图片预加载等。

最近我正在进行网站的响应式优化，使其在移动设备和PC上都有绝佳的浏览体验。同时我正在引入Pinia进行一部分的状态管理，并对现有的代码进行优化。
